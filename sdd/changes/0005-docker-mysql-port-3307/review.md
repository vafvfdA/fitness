# 0005 Docker MySQL 端口调整评审

## 评审摘要

结论：通过，保留容器未实际启动和 Docker CLI 用户配置权限警告的风险。

本变更将本项目 MySQL 的本机端口统一调整为 `3307`，避免和已有 `3306` MySQL 容器冲突。后端默认配置、Docker Compose 和 README 已保持一致。
