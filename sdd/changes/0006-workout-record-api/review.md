# 0006 训练记录后端接口评审

## 评审结论

通过。

本变更完成了训练记录后端的最小闭环，接口能够支撑小程序 A 训练记录页和日历摘要页的初步联调。实现范围控制在训练模块，没有修改既有数据库迁移和无关业务模块。

## 变更摘要

- 新增 `POST /api/v1/workout-records`，用于创建训练记录。
- 新增 `GET /api/v1/workout-records?date=yyyy-MM-dd`，用于查询某天训练记录。
- 新增 `GET /api/v1/workout-records/calendar?month=yyyy-MM`，用于查询月历训练摘要。
- 新增训练记录 DTO、Controller、Service、JDBC Repository。
- 训练总组数由后端根据组明细计算。
- 临时使用 `X-User-Id` 请求头表示当前用户。
- 补充请求头缺失的统一错误响应。
