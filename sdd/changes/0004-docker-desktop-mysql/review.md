# 0004 Docker Desktop MySQL 评审

## 评审摘要

结论：通过，保留 Docker CLI 用户配置权限警告和容器未实际启动的风险。

本变更添加了 Docker Desktop 本地 MySQL 编排配置，并更新 README 说明。配置与后端默认数据库连接保持一致，没有修改业务代码和 Flyway 建表脚本。
